# Config-Bot

## Support Enviornment
Linux / MacOS

## Usecase
1. 下載 config-bot(conb) 安裝
```
curl -OL https://
```
2. 註冊帳號
3. 登入conb
4. $ conb hello-world (會建立一個hello-world專案)

## Structure
online: web(react)
|
online: api(go)
|
local: cli(go)
|
local: api(go)
